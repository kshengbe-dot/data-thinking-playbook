
# Data Thinking Playbook

This repo shows how I approach data problems: from framing questions to preparing data, analyzing patterns, and communicating insights.

## Included
- `index.html` — portfolio-facing data thinking page
- `sql-case-study/` — SQL business case study with schema, sample data, analysis queries, and insights

## SQL Case Study
The SQL case study simulates a retail sales and inventory business scenario. It demonstrates:
- relational database design
- joins, aggregations, grouping, filtering, and subqueries
- revenue and profit analysis
- customer value analysis
- inventory risk detection
- business-oriented recommendations
